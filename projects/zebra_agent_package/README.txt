
Zebra Agent Package

This package contains the local agent that allows a machine to:
1. Detect hardware capability (CPU, RAM, GPU).
2. Scan local directories for topical interests.
3. Query ZebraTube for available tasks.
4. Rank tasks according to suitability.
5. Optionally claim tasks.

Main command:

python zebra-agent/agent.py

Daemon mode:

bash scripts/zebra-agent-start.sh
