# Zebra Studio Complete Scaffold

A larger scaffold for the Zebra ecosystem:

- zebra-core: crawl → extract → graph → projections → scripts
- zebra-agent: local capability / interest scanner and task selector
- zebratube-api: FastAPI backend with task, claim, submission, review, assembly routes
- zebratube-web: React/Vite frontend with homepage, project page, task page, profile page, assembly page

This is still a scaffold, but it is much closer to a full prototype.
