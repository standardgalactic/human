# ZebraTube Monorepo Scaffold

This repository contains a fuller prototype scaffold for:

- `zebra-core` — crawling, extraction, projections, script generation
- `zebra-agent` — local capability + interest scanning
- `zebratube-api` — FastAPI backend
- `zebratube-web` — React/Vite frontend

It is intentionally incomplete but coherent enough to extend into a working prototype.
